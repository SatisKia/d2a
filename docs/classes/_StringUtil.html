クラス _StringUtil

net.satis.d2a._StringUtil

文字列用のユーティリティクラスです。現時点では、decode() メソッドのみです。

メソッド

decode

public static String decode(byte[] sjisBytes, int offset, int length)

指定されたバイト部分配列（Shift-JIS）を復号化して、String オブジェクトを構築します。

パラメータ：
sjisBytes文字列に復号化されるバイト（Shift-JIS）
offset復号化される先頭バイトのインデックス
length復号化するバイト数

戻り値：
構築された String オブジェクト
