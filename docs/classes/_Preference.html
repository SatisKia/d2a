クラス _Preference

net.satis.d2a._Preference

プリファレンスの読み書きの機能を提供するクラスです。

コンストラクタ

_Preference

public _Preference(_Main m, String name)

_Preference コンストラクタです。

パラメータ：
m_Main クラスを継承したクラスオブジェクトを指定します。
name読み書き対象のプリファレンス名を指定します。

メソッド

getParameter

public String getParameter(String key, String defValue)

キー名と値の組み合わせで保存している場合、キーに対応する値を取得します。

パラメータ：
keyキー文字列を指定します。
defValueそのキーの値が存在しないときの初期値を指定します。

戻り値：
キーに対応する値を返します。

setParameter

public void setParameter(String key, String value)

キー名と値の組み合わせでデータを保存します。

パラメータ：
keyキー文字列を指定します。
valueキーに対応する値を指定します。

beginRead

public void beginRead()

データの読み出しを開始します。

read

public String read(String defValue)

データを1個読み出します。

パラメータ：
defValueデータが存在しないときの初期値を指定します。

戻り値：
読み出したデータを返します。

補足説明：
本メソッドは beginRead() から endRead() の中で有効です。

endRead

public void endRead()

データ読み出しを終了します。

beginWrite

public void beginWrite()

データの書き出しを開始します。

write

public void write(String value)

データを1個書き出します。

パラメータ：
value書き出すデータを指定します。

補足説明：
本メソッドは beginWrite() から endWrite() の中で有効です。

endWrite

public void endWrite()

データ書き出しを終了します。
