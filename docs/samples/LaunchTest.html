サンプルアプリ LaunchTest

アプリ連携のデモです。

「LaunchTest」アプリを起動し、画面をタッチしますと、「LaunchSub」アプリの起動を試みます。

プロジェクトの作成

（1）「Eclipse」を起動します。
（2）起動元アプリを用意します。「File」-「New」-「Android Project」を実行し、以下のように設定します。

「Project name」に「LaunchTest」と入力
「Create new project in workspace」が選択されていることを確認
「Build Target」は「Android 2.1」にチェック
「Application name」は「LaunchTest」
「Package name」に「net.satis.launchtest」と入力
「Create Activity」にチェックが入っていることを確認、「LaunchTest」と入力
「Min SDK Version」は「7」

（3）「Finish」ボタンを押します。
（4）起動先アプリを用意します。「File」-「New」-「Android Project」を実行し、以下のように設定します。

「Project name」に「LaunchSub」と入力
「Create new project in workspace」が選択されていることを確認
「Build Target」は「Android 2.1」にチェック
「Application name」は「LaunchSub」
「Package name」に「net.satis.launchsub」と入力
「Create Activity」にチェックが入っていることを確認、「LaunchSub」と入力
「Min SDK Version」は「7」

（5）「Finish」ボタンを押します。
（6）それぞれのプロジェクトにD2Aのソースファイルを追加します。詳細は『プロジェクトの作成』の「（2）D2Aのソースファイルをプロジェクトに追加する」の項を参照ください。

「LaunchTest.java」ファイルの内容を記述する

「LaunchTest」プロジェクトの、「LaunchTest.java」のデフォルトの内容を一旦消去してから、「samples」フォルダの下の「LaunchTest\src\net\satis\launchtest\LaunchTest.java」の内容をコピーしてください。

「LaunchSub.java」ファイルの内容を記述する

「LaunchSub」プロジェクトの、「LaunchSub.java」のデフォルトの内容を一旦消去してから、「samples」フォルダの下の「LaunchSub\src\net\satis\launchsub\LaunchSub.java」の内容をコピーしてください。
