イメージリソースの利用

イメージリソースをアプリケーションに組み込むには、イメージリソース用フォルダを作成してイメージファイルを入れる必要があります。

Package Explorerのプロジェクトの下の「res」をマウス右クリックして、ポップアップメニューの「New」-「Folder」を実行します。

「Folder name」に「drawable-nodpi」と入力し、「Finish」ボタンを押します。

イメージファイルを入れるには、Package Explorerのプロジェクトの下の「res」の中の「drawable-nodpi」にドラッグ＆ドロップするだけです。

組み込まれたイメージリソースを使用する

組み込まれたイメージリソースは、

・_Image クラスの createImage()

メソッドで _Image オブジェクトを生成して使用することができます。例えば次のようになります。

Resources r = getResources();
_Image img_s = _Image.createImage( r, R.drawable.sample );

また、生成された _Image オブジェクトは、不要になったらオブジェクト自身の dispose() メソッドで破棄します。

img_s.dispose();
