開発環境の準備

D2Aを使ってAndroidアプリを作るために必要な開発環境は、標準的なAndroidアプリの開発環境で構いません。

ここでは、Windowsの場合について述べます。他のプラットフォームの場合は、一部の記述を読み替える必要があります。

Eclipse + Android Development Tools + Android SDK

「Eclipse」「Android Development Tools」「Android SDK」がセットになった開発環境です。

http://dl.google.com/android/adt/adt-bundle-windows-x86-20140702.zip
http://dl.google.com/android/adt/adt-bundle-windows-x86_64-20140702.zip（64bit版）

「Java Runtime Environment」を別途インストールする必要があります。

Java Development Kit／Java Runtime Environment

Eclipse（参考）

●ダウンロードサイト
http://www.eclipse.org/downloads/

Android Development Tools（参考）

セットアップ手順
（1）「Eclipse」を起動
（2）「Help」→「Install New Software...」を選択
（3）「Add...」ボタンを押す
（4）「Name」に「Android plug-in」、「Location」に「https://dl-ssl.google.com/android/eclipse/」を入力して「OK」ボタンを押す
（5）ツリーの「Developer Tools」にチェックを入れて「Next >」ボタンを押す
（6）確認画面が表示されるので「Next >」ボタンを押す
（7）「I accept the terms of the license agreements」を選択して「Finish」ボタンを押す

Android SDK（参考）

●ダウンロードサイト
http://developer.android.com/sdk/index.html

セットアップ手順
（1）「android-sdk_r12-windows.zip」をダウンロード
（2）解凍し、フォルダを任意の場所に移動
（3）「Eclipse」を起動
（4）「Window」→「Preferences」を選択
（5）ツリーの「Android」を選択
（6）「SDK Location」に Android SDK のフォルダを入力して「OK」ボタンを押す
（7）「Window」→「Android SDK and AVD Manager」を選択
（8）リストの「Available packages」を選択し、ツリーにチェックを入れ、「Install Selected」ボタンを押す
（9）「Accept All」を選択して「Install」ボタンを押す

Android Virtual Device

セットアップ手順
（1）「Eclipse」を起動
（2）「Window」→「Android SDK and AVD Manager」を選択
（3）リストの「Virtual devices」を選択し、「New...」ボタンを押す
（4）端末の設定を入力し、「Create AVD」ボタンを押す

Android端末のデバッグ用設定

手順（Android 2.x 等）
（1）Android端末の「MENU」ボタンを押す
（2）「端末設定」→「アプリケーション」→「開発」の「USBデバッグ」にチェックを入れる
（3）USB接続モードを「カードリーダーモード」にする

手順（Android 4.x 以降）
（1）「設定」→「端末情報」の「ビルド番号」の欄を複数回タップして、開発者向けオプションを使用可能にする
（2）「設定」→「その他の設定」→「開発者向けオプション」の「USBデバッグ」にチェックを入れる
（3）USB接続モードを「PTPモード」にする

パソコンとAndroid端末のUSB接続

手順
（1）パソコンにAndroid端末をUSBで接続すると、新しいハードウェアの検出ウィザードが表示されるが、キャンセルする
（2）デバイスマネージャを開き、「その他のデバイス」の下の「Android ADB Interface」をダブルクリックし、「詳細」タブでの「ハードウェア ID」をメモしておく
（3）Android SDK のフォルダの下に「extras\google\usb_driver」フォルダがあり、その中の「android_winusb.inf」をテキストエディタで開き、
[Google.NTx86]
の所に記述を追加する

記述例：
%CompositeAdbInterface% = USB_Install, USB\Vid_04dd&Pid_93b4&Rev_0100&MI_01

※ 　　　 メモしておいたハードウェアID

（4）ドライバの更新を行う
（5）パソコンの再起動を要求してきた場合、再起動する
（6）Android端末をUSBで接続すると、デバイスマネージャで「Android Phone」の下に「Android Composite ADB Interface」が表示される
