サウンドリソースの利用

サウンドリソースをアプリケーションに組み込むには、サウンドリソース用フォルダを作成してサウンドファイルを入れる必要があります。

Package Explorerのプロジェクトの下の「res」をマウス右クリックして、ポップアップメニューの「New」-「Folder」を実行します。

「Folder name」に「raw」と入力し、「Finish」ボタンを押します。

サウンドファイルを入れるには、Package Explorerのプロジェクトの下の「res」の中の「raw」にドラッグ＆ドロップするだけです。
